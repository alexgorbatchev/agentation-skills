## Install with skills

List available skills:

```bash
npx skills add alexgorbatchev/agentation-skills --list
```

Install the fix-loop skill for pi:

```bash
npx skills add alexgorbatchev/agentation-skills --skill agentation-fix-loop --agent pi
```
